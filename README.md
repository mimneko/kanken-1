# kanken-1
漢検１級勉強用Webサイト

JSONファイルから問題[^1]を読み取り、ランダムに表として表した。
また、珍しい字の1字フォントまたはsvgファイルは、GlyphWiki[^2]から借りた

K1訓読みの、「まえみつ」「濫用（ランヨウ）」「宛（ずつ）」に「br」が紛れてた。同様に、P7対義語Q7類義語の「欺罔」「大黒」、R8故事成語諺の「麦曲の英」にも。というか対義語と類義語分類できていなくね。

[^2]: https://glyphwiki.org/wiki/GlyphWiki, 参照2024-05-29
